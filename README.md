# Dra Isadora Rosa – Landing Page

Landing Page profissional desenvolvida com **Astro**, **Tailwind CSS** e **TypeScript**, criada para apresentar informações sobre a médica, suas especialidades e facilitar o contato com pacientes.

Este projeto destaca habilidades de frontend moderno, performance, responsividade e SEO.

## Conteúdo da Página

A landing page contém seções organizadas para apresentar a médica, sua abordagem e facilitar o contato com pacientes.

### Seções

- Seção inicial com apresentação da Dra. Isadora e mensagem principal.
- Sobre a médica, contando sua história, missão e abordagem de cuidado.
- Formação acadêmica e certificações profissionais.
- Especialidades em ginecologia e reprodução humana.
- Motivos para escolher a médica, destacando diferenciais.
- Chamada para ação incentivando o agendamento de consultas.
- Seção de contato com links e informações úteis.

## Destaques Técnicos

- Estrutura modular usando componentes Astro.
- Configuração completa de SEO, incluindo OpenGraph e meta tags.
- Deploy automatizado com GitHub Actions e GitHub Pages.
- Design totalmente responsivo para mobile, tablet e desktop.
- Uso de Tailwind com tokens de design personalizados.
- Imagens otimizadas e carregamento eficiente.
- Navegação suave com âncoras e experiência de uso agradável.

## Como rodar o projeto

Todos os comandos devem ser executados na raiz do projeto.

| Comando           | Ação                                                     |
| ----------------- | -------------------------------------------------------- |
| `npm install`     | Instala as dependências                                  |
| `npm run dev`     | Inicia o servidor de desenvolvimento em `localhost:4321` |
| `npm run build`   | Gera a versão de produção em `./dist/`                   |
| `npm run preview` | Faz um preview local da versão de produção               |
| `npm run check`   | Verifica erros no projeto                                |
| `npm run fix`     | Formata o projeto com ESLint e Prettier                  |
